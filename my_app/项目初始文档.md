项目路径： my_app

技术栈： Vue 3.3 + TypeScript + Vite 5 + Vue Router 4 + Vue I18n 9 + Axios

目录结构：


my_app/
├── src/
│   ├── api/
│   │   ├── auth.ts          # 认证接口
│   │   └── request.ts       # Axios实例+拦截器，baseURL: http://www.v1.com
│   ├── components/
│   │   └── PopupModal.vue   # 全局弹窗UI组件
│   ├── composables/
│   │   └── usePopup.ts      # 弹窗状态管理（模块级单例）
│   ├── config/
│   │   └── popupPresets.ts  # 弹窗预设（success/error/warning/info）
│   ├── hooks/
│   │   └── useAuth.ts       # 登录/登出/token管理
│   ├── locales/
│   │   ├── zh.json          # 中文
│   │   └── en.json          # 英文
│   ├── router/
│   │   └── index.ts         # 路由+beforeEach权限守卫
│   ├── views/
│   │   ├── Home.vue         # 首页
│   │   └── Login.vue        # 登录页
│   ├── App.vue              # 根组件（含全局PopupModal）
│   └── main.ts              # 入口，挂载Vue+Router+i18n
└── package.json
